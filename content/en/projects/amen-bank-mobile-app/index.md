<h2 class="text-3xl font-bold text-text-primary dark:text-text-primary mb-6 flex items-center gap-3">
<span class="w-8 h-8 rounded-full bg-primary/20 text-primary flex items-center justify-center text-sm"><i class="fas fa-layer-group"></i></span>
Clean Architecture Implementation
</h2>
<div class="bg-bg-secondary dark:bg-bg-secondary border border-border-primary dark:border-border-primary rounded-2xl p-6 md:p-8 shadow-sm">
<p class="text-text-secondary dark:text-text-secondary mb-6 leading-relaxed">To ensure maintainability and scalability for a major banking application, the mobile app was built using a strict clean architecture approach, separating concerns into distinct layers.</p>

<div class="space-y-4">
<div class="flex items-start gap-4 p-4 rounded-xl bg-bg dark:bg-bg border border-border-primary">
<div class="w-10 h-10 rounded-full bg-primary/10 flex items-center justify-center shrink-0">
<i class="fas fa-mobile-alt text-primary text-xl"></i>
</div>
<div>
<h4 class="font-bold text-text-primary">Presentation Layer</h4>
<p class="text-sm text-text-secondary mt-1">Handles the UI and state management using modern Flutter patterns, providing a highly responsive and fluid user experience optimized for complex banking operations.</p>
</div>
</div>
<div class="flex items-start gap-4 p-4 rounded-xl bg-bg dark:bg-bg border border-border-primary">
<div class="w-10 h-10 rounded-full bg-primary/10 flex items-center justify-center shrink-0">
<i class="fas fa-briefcase text-primary text-xl"></i>
</div>
<div>
<h4 class="font-bold text-text-primary">Domain Layer</h4>
<p class="text-sm text-text-secondary mt-1">Encapsulates the core business logic and entities, entirely independent of the UI and data sources, ensuring critical banking rules remain pure and testable.</p>
</div>
</div>
<div class="flex items-start gap-4 p-4 rounded-xl bg-bg dark:bg-bg border border-border-primary">
<div class="w-10 h-10 rounded-full bg-primary/10 flex items-center justify-center shrink-0">
<i class="fas fa-database text-primary text-xl"></i>
</div>
<div>
<h4 class="font-bold text-text-primary">Data Layer</h4>
<p class="text-sm text-text-secondary mt-1">Manages data retrieval and storage, securely connecting to backend banking APIs via encrypted channels and managing local caching for swift offline capabilities.</p>
</div>
</div>
</div>
