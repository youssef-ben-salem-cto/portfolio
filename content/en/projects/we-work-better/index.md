<h3 class="text-3xl font-extrabold text-text-primary mb-4">Enterprise Architecture Deep Dive</h3>
<p class="text-text-secondary max-w-2xl mx-auto text-lg">A sophisticated orchestration of modern technologies designed to guarantee high availability, strict data governance, and seamless tenant isolation.</p>
</div>

<div class="grid grid-cols-1 md:grid-cols-2 gap-6 mb-12">

<i class="fas fa-project-diagram text-primary text-2xl"></i>
</div>
<h4 class="text-xl font-bold text-text-primary mb-3">Nest.js Bounded Contexts</h4>
<p class="text-text-secondary leading-relaxed">The backend is architected using strict Domain-Driven Design (DDD) principles. By isolating business logic into explicit bounded contexts, the system allows autonomous teams to scale individual microservices independently without risking domain leakage.</p>
</div>

<div class="p-8 rounded-2xl bg-bg-secondary border border-border-primary hover:border-brand-primary/50 transition-colors group">
<div class="w-14 h-14 rounded-xl bg-primary/10 flex items-center justify-center mb-6 group-hover:scale-110 transition-transform">
<i class="fas fa-stream text-primary text-2xl"></i>
</div>
<h4 class="text-xl font-bold text-text-primary mb-3">Event-Driven Workflow via Kafka</h4>
<p class="text-text-secondary leading-relaxed">Core business operations are decoupled using Apache Kafka. This event-driven architecture guarantees eventual consistency, robust message replayability, and peak load handling for mission-critical B2B integrations.</p>
</div>

<div class="p-8 rounded-2xl bg-bg-secondary border border-border-primary hover:border-brand-primary/50 transition-colors group">
<div class="w-14 h-14 rounded-xl bg-primary/10 flex items-center justify-center mb-6 group-hover:scale-110 transition-transform">
<i class="fas fa-shield-alt text-primary text-2xl"></i>
</div>
<h4 class="text-xl font-bold text-text-primary mb-3">Zero-Trust Identity with Authentik</h4>
<p class="text-text-secondary leading-relaxed">Enterprise IAM is centralized via Authentik, enabling seamless SSO, rigorous Role-Based Access Control (RBAC), and multi-factor authentication to meet strict corporate compliance standards.</p>
</div>

<div class="p-8 rounded-2xl bg-bg-secondary border border-border-primary hover:border-brand-primary/50 transition-colors group">
<div class="w-14 h-14 rounded-xl bg-primary/10 flex items-center justify-center mb-6 group-hover:scale-110 transition-transform">
<i class="fas fa-database text-primary text-2xl"></i>
</div>
<h4 class="text-xl font-bold text-text-primary mb-3">Hybrid Persistence Strategy</h4>
<p class="text-text-secondary leading-relaxed">Data modeling leverages both PostgreSQL for strict ACID-compliant relational data and MongoDB for flexible document storage. High-speed caching is powered by Redis, while Minio (with a custom presigner service) manages secure S3-compatible object storage.</p>
</div>
</div>
