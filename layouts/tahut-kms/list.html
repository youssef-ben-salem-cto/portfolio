Vision & Roadmap

Phase 1: Precision & Proof

Completed

Tahut core integration, IDE-style Workspace, Sub-200ms Citation Snapping, Migration to OpenSearch/pgvector and Neo4j, with unstructured and docling support.

Phase 2: Growth & Governance

Completed

Enterprise Identity Federation (SAML/OIDC), Headless API Integration (Expose Pyramids via JWT), Cognitive Agent Layer (ReAct/CoT), Cross-device Chat Session Sync, Distributed Worker scaling.

Phase 3: The Intelligence OS

Current
  • Unified AI Storage Expansion: Native support for MongoDB as an alternative all-in-one operational, vector, and document persistence engine.
  • Knowledge Forking: Branch and clone specific Pyramids/Collections for isolated experimentation and reasoning workflows.
  • Collection Composer Orchestrator: Treating distinct knowledge bases as tools—dispatching parallel agentic retrieval tasks and synthesizing unified answers via cognitive merging.
  • Predictive Prompting: Dynamic, context-aware generation of AI-recommended follow-up queries based on the ongoing reasoning stream.

Phase 4: "Eye of Horus" Alerts

Future
  • Active Intelligence Layer: Transforming the passive KMS repository into a dynamic, real-time monitoring and alerting engine.
  • Declarative Rules Engine: Configuration-driven graph pattern matching via Neo4j Cypher and APOC triggers.
  • Dual-Path Alert Streaming: Instant persistence to local dashboard alongside Kafka topic distribution for SIEM integration.

Future

  • Modular Logic Engine: Configuration-driven rule and query processors for tailored, proactive recommendations and natural language responses.
